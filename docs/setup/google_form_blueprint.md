# Google Form 질문 구성표

## 폼 제목

예약 접수 폼

## 폼 설명

```text
예약 안내 및 변경 안내를 위해 필요한 최소 정보를 입력해주세요.
입력하신 정보는 예약 관리와 안내 목적으로만 사용됩니다.
```

## 기본 질문

| 순서 | 질문 | 유형 | 필수 | 시트 매핑 | 비고 |
|---:|---|---|---|---|---|
| 1 | 이름 | 단답형 | 예 | 고객명 | 실명 또는 예약명 |
| 2 | 연락처 | 단답형 | 예 | 연락처 | 010-0000-0000 형식 안내 |
| 3 | 예약 희망일 | 날짜 | 예 | 예약일 | 날짜 형식 |
| 4 | 예약 희망시간 | 시간 | 예 | 예약시간 | 시간 형식 |
| 5 | 예약 항목 | 드롭다운 | 예 | 서비스명 | 업종별 변경 |
| 6 | 담당자 | 드롭다운 | 선택 | 담당자 | 내부 입력용이면 생략 가능 |
| 7 | 유입 경로 | 객관식 | 선택 | 유입경로 | 네이버/전화/소개/홈페이지/기타 |
| 8 | 요청사항 | 장문형 | 선택 | 메모 | 민감정보 입력 금지 안내 |
| 9 | 예약 안내 메시지 수신 동의 | 체크박스 | 예 | 수신동의 | 동의해야 제출 가능하게 설정 권장 |

## 수신 동의 문구

```text
예약 확정, 변경, 전날 안내, 당일 안내를 위해 문자 또는 카카오톡으로 연락받는 것에 동의합니다.
```

## 요청사항 안내 문구

```text
요청사항에는 예약과 방문에 필요한 내용만 적어주세요.
진료 내용, 질병명, 검사 결과, 민감한 개인정보는 입력하지 마세요.
```

## Google Sheets 매핑 권장 구조

Google Form 응답은 먼저 `Form_Responses` 시트에 쌓이게 두고, Apps Script 또는 수식으로 `Appointments` 시트에 정리하는 방식을 추천합니다.

| Form_Responses | Appointments |
|---|---|
| 타임스탬프 | 생성일 |
| 이름 | 고객명 |
| 연락처 | 연락처 |
| 예약 희망일 | 예약일 |
| 예약 희망시간 | 예약시간 |
| 예약 항목 | 서비스명 |
| 담당자 | 담당자 |
| 유입 경로 | 유입경로 |
| 요청사항 | 메모 |
| 수신 동의 | 수신동의 |

## 폼 설정 체크리스트

- 응답을 Google Sheets에 연결한다.
- 이메일 주소 자동 수집은 고객 상황에 따라 선택한다.
- 응답 수정 허용은 기본적으로 끈다.
- 제출 후 확인 메시지를 설정한다.
- 민감정보 입력 금지 문구를 요청사항 위에 넣는다.
- 테스트 응답 3건 이상을 넣어 시트 연결을 확인한다.

## 제출 후 확인 메시지

```text
예약 요청이 접수되었습니다.
담당자가 확인 후 예약 확정 안내를 드리겠습니다.
예약 변경이 필요하시면 사업장으로 직접 연락해주세요.
```
