# 개인정보 수집·이용 안내 문구 템플릿

## 사용 목적

Google Form 하단 또는 예약 접수 안내문에 넣을 수 있는 개인정보 수집·이용 안내 문구입니다.

법률 자문 문서가 아니므로 실제 고객에게 납품할 때는 고객 업종, 수집 항목, 보관기간, 내부 개인정보 처리방침에 맞게 조정해야 합니다.

## 기본형

```text
[개인정보 수집·이용 안내]

1. 수집 항목
- 이름
- 연락처
- 예약 희망일
- 예약 희망시간
- 예약 항목
- 예약 안내 수신 동의 여부

2. 수집·이용 목적
- 예약 접수 및 확인
- 예약 변경 안내
- 전날/당일 예약 안내
- 취소 또는 미방문 후속 안내
- 예약 운영 현황 관리

3. 보관기간
- 예약일로부터 1년간 보관 후 삭제 또는 마스킹합니다.
- 단, 관련 법령 또는 내부 분쟁 대응을 위해 필요한 경우 정해진 기간 동안 보관할 수 있습니다.

4. 동의 거부 권리
- 개인정보 수집·이용에 동의하지 않을 수 있습니다.
- 다만 동의하지 않는 경우 예약 안내 메시지 발송이 제한될 수 있습니다.

5. 민감정보 입력 금지
- 요청사항에는 예약과 방문에 필요한 내용만 입력해주세요.
- 질병명, 진료 내용, 검사 결과, 주민등록번호, 결제정보 등 민감한 정보는 입력하지 마세요.
```

## 짧은 폼용 문구

```text
예약 확인과 안내를 위해 이름, 연락처, 예약일, 예약시간, 예약 항목을 수집합니다.
수집된 정보는 예약 관리와 안내 목적으로만 사용되며, 예약일로부터 1년 후 삭제 또는 마스킹됩니다.
요청사항에는 질병명, 진료 내용, 검사 결과, 주민등록번호, 결제정보 등 민감한 정보를 입력하지 마세요.
```

## 체크박스 동의 문구

```text
위 개인정보 수집·이용 안내를 확인했으며, 예약 확인 및 안내 메시지 수신에 동의합니다.
```

## 직원용 개인정보 처리 원칙

```text
예약 관리 시트에는 예약 안내에 필요한 최소 정보만 입력합니다.
진료 내용, 질병명, 검사 결과, 주민등록번호, 결제정보는 입력하지 않습니다.
고객 삭제 요청이 있으면 담당자에게 즉시 알리고, 원본 시트와 백업 시트 모두 확인합니다.
```
