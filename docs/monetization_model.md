# Free Template + Paid Operations Monetization Model

## Strategy

Make the template free and public.

Earn revenue from services that busy businesses do not want to handle themselves:

- Setup
- Customization
- Security review
- Staff training
- Monthly operation
- Reporting
- Integrations

The free template builds trust. Paid services remove friction.

## Free Offer

Public GitHub repository:

- Google Sheets template
- Apps Script automation
- Google Form guide
- Privacy notice template
- Security checklist
- QA test cases
- Recovery plan

Positioning:

```text
Free Google Sheets reservation reminder automation for small appointment-based businesses.
```

## Paid Service Ladder

| Stage | Offer | Price Idea | Goal |
|---|---:|---:|---|
| Free | GitHub template | Free | Build trust and inbound leads |
| Setup | Basic installation | 300,000~700,000 KRW | First paid conversion |

## Why People Pay If The Template Is Free

Most businesses do not pay for the file.

They pay for:

- Not having to understand the setup
- Avoiding mistakes with customer data
- Having someone customize messages
- Having someone test edge cases
- Having someone answer staff questions
- Having a monthly operator

The free project is the proof. The service is the product.

## Conversion Funnel

1. GitHub visitor downloads template.
2. README explains setup and security.
3. Visitor sees paid setup option.
4. Visitor books free 15-minute workflow diagnosis.
5. You identify one painful workflow.
6. Offer setup or standard package.
7. Convert to monthly maintenance after 2 weeks.
