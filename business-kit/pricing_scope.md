# 가격표와 작업 범위

## 가격 원칙

초기에는 `템플릿 판매`가 아니라 `설치형 자동화 구축`으로 판매합니다. 고객은 파일 자체보다 세팅, 문구 정리, 사용법 교육, 오류 대응에 비용을 지불합니다.

## 추천 패키지

| 패키지 | 가격 | 추천 대상 | 포함 범위 |
|---|---:|---|---|
| Basic | 30만~70만 원 | 아주 작은 매장, 첫 포트폴리오 고객 | 예약 폼, 예약 시트, 기본 문구 생성 |
| Standard | 80만~150만 원 | 병원, 학원, 샵 등 일반 예약제 사업장 | Basic + 발송 대기함 + 미방문 관리 + 주간 리포트 |
| Premium | 150만~300만 원 | 예약량이 많고 직원이 여러 명인 사업장 | Standard + 업종별 맞춤 문구 + Apps Script 자동화 + 보안 점검 + 교육 |
| 유지보수 | 월 10만~50만 원 | 계속 수정/관리 원하는 고객 | 문구 수정, 오류 점검, 월간 리포트, 간단 개선 |

## Basic 상세 범위

- Google Form 예약 접수 폼 1개
- Google Sheets 예약 목록
- 예약확정 문구 템플릿
- 전날안내 문구 템플릿
- 기본 사용법 안내

제외:

- Apps Script 자동화
- 주간 리포트
- 문자/카카오 자동 발송
- 복잡한 권한 관리

## Standard 상세 범위

- Google Form 예약 접수 폼 1개
- Google Sheets 예약관리 대시보드
- 예약확정, 전날안내, 당일안내, 미방문안내, 리뷰요청 문구
- 발송 대기함
- 미방문 고객 후속 관리
- 주간 예약 리포트
- 직원용 사용 안내서
- 샘플 데이터 정리
- 개인정보 수집·이용 안내 문구
- 기본 보안 체크리스트

제외:

- 외부 API 자동 발송
- EMR/POS/CRM 직접 연동
- 고객 로그인 페이지
- 결제 기능
- 전문 상담/진료 판단 자동화

## Premium 상세 범위

- Standard 전체
- 업종별 맞춤 문구 5종 이상
- 담당자별 필터
- 유입 경로 분석
- Apps Script 메뉴 설치
- 발송 대기함 자동 생성
- 주간 리포트 자동 갱신
- 직원 교육 1회
- 보안 점검 메뉴 설치
- 마스킹 뷰 생성
- 백업/복구 절차 세팅
- 보호 설정 적용

옵션:

- 문자 API 연동
- 카카오 알림톡 연동
- 이메일 자동 발송
- 월간 PDF 리포트
- 웹 대시보드 전환
- Google Workspace 관리자 보안 설정 컨설팅

| API 연동 | 별도 협의 |

## 계약 전 확인할 것

- 예약 건수는 하루 평균 몇 건인가?
- 예약 접수 채널은 무엇인가?
- 전날/당일 안내는 누가 보내는가?
- 문자/카카오 발송 권한은 누가 갖고 있는가?
- 고객정보 수집 동의 문구가 있는가?
- 직원이 Google Sheets를 사용할 수 있는가?
- 기존 예약 데이터를 옮겨야 하는가?

## 범위 초과 예시

아래 요청은 별도 견적으로 분리합니다.

- 기존 시스템 직접 연동
- 자동 문자 대량 발송
- 카카오 알림톡 심사/템플릿 등록 대행
- 복잡한 예약 캘린더 구현
- 고객용 로그인 페이지
- 의료/법률/세무 판단 자동화
- 결제, 환불, 정산 기능
- 다지점 통합 관리
